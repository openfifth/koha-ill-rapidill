# koha-ill-rapidill
A Koha Interlibrary Loans (ILL) backend for RapidILL

## Getting Started

This plugin requires the following Perl modules:

[XML::Compile](https://metacpan.org/pod/XML::Compile)

[XML::Compile::SOAP]()

[XML::Compile::WSDL11](https://metacpan.org/pod/XML::Compile::WSDL11)

[XML::Compile::SOAP12](https://metacpan.org/pod/XML::Compile::SOAP12)

ktd installation:
$ sudo cpan XML::Compile XML::Compile:WSDL11 XML::Compile::SOAP12

Download this plugin by downloading the latest release .kpz file from the [releases page](https://github.com/openfifth/koha-ill-rapidill/releases).
